Cloudflare security checklist

Change nameservers

Use the two Cloudflare nameservers at your registrar and wait until Cloudflare shows Active.

GitHub Pages custom domain

Set custom domain to epictech.club and commit the CNAME file included in this site.

SSL/TLS

Use Full for GitHub Pages. Enable Always Use HTTPS and Automatic HTTPS Rewrites.

WAF and bot basics

Enable managed WAF rules where available, bot protection, and rate limiting for future dynamic portals.

Keep launch static

No database, no WordPress admin, no exposed server. Add dynamic portals later only when revenue justifies the risk.
